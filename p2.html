This Game is NOT Working! 2

In this sketch I played with different kinds of built-in function, array fuctions, images, and gif functions. I initially started my project by creating a 2d shooting game, I used "let" function to build an array of things like bullets, enemies, score systems, and many more.

I then included preload images and gifs in the background by using functions like boolean logic, loadImage(), image(), tint(), and preload.

The most challenging part of this project was making the game work, but I spent way too much time on it that I lost track of what I actually need to work on for this excercise. I then change my focus on figuring out how to put images and text for this excercise. Also, I just try to make the most out of what I currently know and not overthink the porject too much.

This is the excercise where I had the most fun. Eventhough I tried to create a game that does not work properly, I learned a lot of things along the way and I get to use my creativity to make something out of a broken game.
